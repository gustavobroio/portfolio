Habilidades

HTML


O HyperText Markup Language, ou HTML, é a linguagem de marcação padrão para documentos projetados para serem exibidos em um navegador da web.

Pode ser auxiliado por tecnologias como Cascading Style Sheets (CSS) e linguagens de script como JavaScript.

CSS


Cascading Style Sheets (CSS) é uma linguagem de folha de estilo usada para descrever a apresentação de um documento escrito em uma linguagem de marcação como HTML.

O CSS é projetado para permitir a separação entre apresentação e conteúdo, incluindo layout, cores e fontes.

JavaScript


JavaScript, frequentemente abreviado como JS, é uma linguagem de programação que é uma das tecnologias principais da World Wide Web, juntamente com HTML e CSS.

Mais de 97% dos sites utilizam JavaScript no lado do cliente para o comportamento das páginas da web, muitas vezes incorporando bibliotecas de terceiros.
